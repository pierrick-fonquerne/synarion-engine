# Synarion Engine

**Synarion Engine** is a professional open-source 3D game engine written in Rust with Vulkan.

## Goals

- **Performance**: Native Rust performance with Vulkan graphics
- **Modularity**: ~55 independent crates for maximum flexibility
- **Developer Experience**: Professional editor with modern UX
- **Open Source**: MIT licensed, community-driven development

## Features (Planned)

- Vulkan-based rendering (PBR, shadows, post-processing)
- Entity Component System (ECS)
- Physics (via Rapier)
- Audio system
- Networking (client/server)
- Procedural generation
- Full-featured editor
- Lua scripting

## Status

Synarion Engine is currently in **early development**. We are building the foundation crates.

See the [Roadmap](./roadmap/status.md) for current progress.
